# weather

Do you want to know the current weather conditions?

```
weather
```
:point_down:
```
The current temperature in Tokyo is 0.4° C and the sky is light rain.
Have a nice day!
```

```
weather locate Oslo
```
:point_down:
```
The current temperature in Oslo is 2.4° C and the sky is light rain.
Have a nice day!
```

Good luck!
